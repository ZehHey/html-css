Representando cores em CSS3

É possível representar cores em css de várias maneiras

Independente do modo escolhido para representar uma cor, o Visual Studio Code, ajuda a selecionar as diversas variações dessa cor, basta passar o mouse sobre o nome ou código da cor, e o próprio programa irá mostrar uma janela seletora de cores, com possibilidade de configurar até mesmo a transparência.
