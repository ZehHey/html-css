Testando Gradiente

Para colocar degradê na cor de fundo da página, dentro da área <head> coloca-se a tag <style> e então o parâmetro body{} e substitui o background-color por background-image: linear-gradient.

No gradiente linear se define posição e quantidade de cores

        
        <style>
        *{ >> configuração global
            
        }
        body {
            background-image: linear-gradient(to top, white,blue);
            background-attachment: fixed**
        }
        </style>

As direções do gradiente são representadas por:

  • to right - para a direita
  • to left - para a esquerda
  • to top - para cima
  • to bottom - para baixo

Ou podem ser representadas por grau, por exemplo:

  • 90deg - 90°
  • -45deg - 45° negativo

É possível substituir o parâmetro linear-gradient por radial-gradiente com a direção definida como circle.

O parêmetro background-attachment: fixed; permita que o fundo fique fixo mesmo rolando a página.
