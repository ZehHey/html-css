Trabalhando com fontes

Na tag <style> define-se a font-family, onde geralmente e de forma recomendável, define-se três parâmetros. O primeiro é a fonte que se deseja utilizar, caso o dispositivo não possua essa fonte instalada, ele irá utilizar outra fonte, que é o segundo parâmetro. O terceiro parâmetro é a categoria da fonte, onde o dispositivo irá selecionar uma fonte pertecente a essa categoria, caso não consiga exibir nenhum dos parâmetros anteriores.

        
            <style>
                body{
                    font-family: Arial, Helvetica, sans-serif;
                }
                h1, h2 {
                    font-family: 'Times New Roman', Times, serif;
                }
                </style>

São as chamadas Web safe font combinations.
