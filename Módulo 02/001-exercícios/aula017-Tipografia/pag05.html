Tipos de alinhamento

Subtitulo 1

Existem três tipos de alinhamentos básicos: centralizado, à direita e à esquerda. Para definir o alinhamento usa-se o parâmetro "text-align:" no style local ou global.

Pode-se usar o parâmetro "text-indent:" para criar um recuo (uma indentação) no início de cada parágrafo.

Exemplo desta página:

            <style>
                h1{
                    text-align: center;
                }
                h2{
                    text-align: right;
                }
                p{
                    text-align: justify;
                    text-indent: 30px;
                }
            </style>
