Google Fonts

O próprio Google Fonts já nos fornece o código para incorporar a fonte desejada no código do site que está sendo criado, com o @import.


Fontes externas baixadas

No caso da necessidade de baixar um fonte, os arquivos .ttf e .otf são salvos no servidor da página. E então é criado em CSS o @font-face, que possibilitará o uso da fonte no código criado.

Ex: 

            
                @font-face {
                    font-family: 'StarJedi';    --> nome dado à familia da fonte
                    src: url(fontes/Starjedi.ttf) format(truetype);
                }

Tipos de format():

  • opentype (opt);
  • truetype (ttp);
  • embedded-opentype;
  • truetype-aat (Apple Advanced Typography);
  • svg;

Capturando Fontes de um site

Para isso pode-se usar a extensão Fonts Ninja no Chrome.


Detectando fontes dentro de imagens

São três sites que oferecem um ferramenta para adetecção de fontes em imagens. Obs: Não é 100% garantido que funcione em todos os casos.

Os sites são:

  • What Font Is.
  • Font Squirrel.
  • My fonts.
