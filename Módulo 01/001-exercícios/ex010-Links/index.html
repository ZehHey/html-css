Usando Links Externos.

Usa-se a marcação <a> e no parâmetro href que aparece no código, coloca-se a url do site/página desejado.

No caso de links externos, recomenda-se sempre usar os parâmetros target="_blank" e rel="external", para o link abrir em uma nova aba.

Usando Links Internos

Link interno faz relação com outra página que está dentro do mesmo servidor.

Links para Download

Dentro da marcação <a> no parâmetro href põe-se o arquivo a ser baixado, juntamente com o parâmetro download="Nome do arquivo" e o parâmetro type

  • Livro em PDF.- (O navegador está abrindo o arquivo em PDF.)
  • Livro em RAR.

Obs: segue o link com a lista type para cada tipo de arquivo.
