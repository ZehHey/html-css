Trabalhando com Listas

Listas Ordenadas (OL)

Usa-se a marcação <ol> e pode-se alterar entre os tipos de marcadores da lista (1, A, a, I, i) usando o parâmetro type, sendo "1" o padrão.
  1. Acordar
  2. Ligar para João
  3. Tomar café
  4. Escovar os dentes
  5. Ir pra faculdade
  6. Almoçar
É possível alteral o número de início da lista com o parâmetro start que é um padrão numérico.

Listas não Ordenadas

Usa-se a marcação <ul> que possui três opções de type: disc (padrâo), circle e square
  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Alface
  • Arroz

Lista de Definições

Marcação <dl>. É como um dicionário, coloca-se um termo (<dt>) e sua definição (<dd>).
Ex:
HTML
Linguagem de marcação para a criação de conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.
