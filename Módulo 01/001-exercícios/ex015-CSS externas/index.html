Capítulo 1.1

Nesse método, cria-se um arquivo.css exclusivo para o estilo, e em cada página do site na parte de head cria-se um link:css para esse arquivo.

O modo mais rápido é criando o link primeiro, então segura o ctrl e clica em style.css, assim o arquivo de estilo será criado automaticamente e é nesse arquivo que será colocada a formatação e estilo para as páginas do site.

É recomendável usar a regra charset no arquivo css, para evitar problemas com acentuações e outros caracteres, assim como nas páginas html, para isso basta digitar @charset "UTF-8"; na primeira linha.

Capítulo 1.2 - Disposição do arquivo CSS

No arquivo de estilo "style.css" os seletores e declarações ficam do seguinte modo:

        
            @charset "UTF-8";

            body{
                background-color: black;
                color: aliceblue;
                font-family: Arial, Helvetica, sans-serif;
                font-size: 17px;
                }
            h1{
                color: black;
                background-color: lightblue;
              }
            h2{
                color: darkred;
            }
            p{
                text-align: justify;
             }
