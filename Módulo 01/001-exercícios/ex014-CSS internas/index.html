Capítulo 1.1 - Aplicação

Nesse caso a tag <style> é colocada na parte de head da HTML. Esse método é baseado em seletores. Dentro da tag <style> crio o seletor para a parte do código que será estilizada e então coloco a declaração:

        
            body{
                background-color: lightsteelblue;
                font-family: Arial, Helvetica, sans-serif;
                font-size: 17px;
                }
            h1{
                color: darkblue;
                }

Capítulo 1.2 - Desvantagens

Uma das desvantagens desse método é que o style ocupa mais linhas do que o conteúdo propriamente dito.

A outra desvantagem é que a estilização não será aplicada às outras páginas do site criado. No caso seria necessário copiar os seletores e declarações em todas as outras páginas para manter o padrão.
