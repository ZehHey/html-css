Videos em hospedagem própria.

Os diferentes navegadores são compatíveis com determinados formatos, por isso o ideal é ter o mesmo vídeo em formatos diferentes.

Este vídeo está hospedado no meu próprio servidor

Dentro da tag <video controls> coloca-se tantas tags <source:src> necessárias para a quantidade de formatos diferentes criados do mesmo vídeo

Após a tag controls foi definida também a tag width em 560px para reduzir o tamanho da tela do vídeo no site.

É possível também colocar uma imagem como capa do vídeo com a tag poster depois da tag width.

Essa forma de reprodução de vídeo com hospedagem própria, pode ser problemática dependendo do número de acessos no site, pois consomem muito tráfego de dados, oque pode acarretar no encarecimento do serviço de hospedagem contratado.


Incorporação de vídeos externos

Vídeos do Youtube.

Dentro da opção Compartilhar em algum vídeo do Youtube, exite a opção Incorporar, na qual copia-se o código gerado e cola dentro do código HTML do seu site.

Essa é a forma de incorporação de vídeo mais indicada pois o tráfego de dados ficará por conta do site externo, nesse caso o Youtube.


Outra opção para incorporação de vídeos externos é o Vimeo.
