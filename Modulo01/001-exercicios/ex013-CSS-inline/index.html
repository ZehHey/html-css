Capítulo 1.1

As CSS são folhas de estilo, e podem ser aplicadas de três formas. A primeira forma é o método inline é o menos recomendado, mas serve como primeiro contato. Esse método é mais indicado para configurações pontuais e pequenas.

Dentro da tag <body> ou <h1>, etc, coloca-se o parâmetro style e outros parâmetros como: background-color ou color.

Percebe-se que dessa maneira não fica padronizado todos os h1 com a mesma cor. Isso teria que ser feito manualmente. Outro ponto negativo do estilo inline é que ele polui visualmente o seu código HTML.
