Pseudo-Classes

As pseudo-classes estão relacionadas à configuração do estado de um determinado elemento, por exemplo, se esse elemento está ativo, se está marcado, etc. Elas são identificadas por ":".

No exemplo abaixo, os elementos foram envelopados dentro de uma <div> e foram configuradas pseudo-classes "hover" nos parâmetros <h3> para que as letras fiquem vermelhas e para que os parâgrafos sejam exibidos quando o ponteiro do mouse passa sobre elas.

            <style>
            div > p{
                display: none;      >>> esconder o parágrafo
            }
            div:hover > p{
                display: block;     >>> mostrar o parágrafo
            }
            h3:hover{
                color: red;         >>> deixar a letra vermelha
            }
            </style>

Pseudo-Elementos

Os pseudo-elementos agem no contéudo do elemento, e podem agir nas classes, ids, e elementos. São identificados por "::".

No exemplo abaixo foi adicionado o símbolo ✎ após os títulos <h3>.

            h3::after{
                content:' ✎';
            }

*Símbolos Usados

  • #  =  id
  • .  =  class
  • :  =  pseudo-class
  • ::  =  pseudo-element
