Tudo o que é exibido na tela, é exibido em forma de caixa.

Anatomia de uma caixa


Tipos de caixa

Box-Level (Block):

São caixas que sempre iniciam em uma nova linha, e ocupam a largura total da tela (viewport)

Principais Tags:

<div>

<h1> - <h6>

<p>

<main>

<header>

<nav>

<article>

<aside>

<footer>

<form>

<video>

Inline-Level:

São caixas que não se iniciam em uma nova linha, ocupam a largura necessária para conter o conteúdo, e no final também não quebram a linha.

Principais Tags:

<span>

<a>

<code>

<small>

<strong>

<em>

<sup> - <sub>

<label>

<button>

<input>

<select>
É possível transformar uma box-level em uma inline-level e vice-versa com o parâmetro display.

Grouping Tags

As grouping tags existem para que se monte um site de forma semântica.

  • <header> - cabeçalho
  • <main> - conteúdo principal
  • <footer> - rodapé
  • <nav> - menu
  • <section> - sessão
  • <aside> - conteúdo periférico
  • <article> - artigo
