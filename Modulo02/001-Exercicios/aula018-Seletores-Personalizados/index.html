Seletor personalizado

Em que momento usar:

Quando temos dois elemento iguais em um documento HTML, mas que precisam ter formatações diferentes.

Como usar:

Coloca-se o parâmetro "id=" dentro do elemento desejado e dá-se um nome a esse elemento para diferenciá-lo dos demais. Ex:

            <h1 id="principal">Exemplos</h1>
            <h2 id="h2html">Criando um site com HTML e CSS</h2>
            <h2 id="h2css">Aprendendo CSS/<h2>

Então na página de configuração de estilo,configura-se esse elemento pelo nome associado a ele. Para identificá-lo usa-se "#".Ex:

            #principal{
                background-color: white;
                color: rgb(6, 6, 119);
                text-align: center;
            }
            #h2html{
                background-color: rgb(48, 56, 95);
                color: aliceblue;
            }
            
            #h2css{
                background-color: rgb(84, 31, 134);
                color: rgb(209, 209, 216);
            }

Usando classes

Quando se tem dois ou mais elementos que precisam ter a mesma formatação dentro do mesmo documento HTML, usa-se o parâmetro class, que também faz a configuração personalizada.

É recomendável que não se nomeie as classes pela sua forma (ex: verde), mas sim por sua função (ex: básico).

            <h3 class="basico">HTML básico</h3>
            <h3 class="basico">CSS básico</h3>

As classes não são identificadas com "#" nas folhas de estilo, e sim com um ".". 

   
            .basico{
                color: rgb(102, 102, 211);
            }

            .intermediario{
                color: rgb(46, 46, 126);
            }

É possível usar uma mesma classe no meio de um parágrafo também, basta envelopar o trecho com a tag <span> e usar o parâmetro class.

            (...)trecho com a tag <span> e usar o parâmetro <span 
            class="basico">class</span>
Pode-se também usar mais de uma classe em um mesmo elemento, além de um id e class observando sempre a hierarquia das configurações: 
            (...)classe em um mesmo <span 
            class="intermediario destaque">elemento</span>
